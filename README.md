# file-names-normalizer

## Overview

This project contains a Bash script designed for renaming files in a directory in a standardized way. It works across different operating systems (macOS and Linux) and modifies file names by performing the following actions:

Converting file names to lowercase.
Replacing spaces in file names with underscores.
Ensuring compatibility with both macOS and Linux file listing commands.
